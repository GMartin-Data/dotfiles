# Niveaux de guidage

## Les 4 niveaux

| Niveau | Nom | Quand | Claude fournit | Claude NE fournit PAS |
|--------|-----|-------|----------------|----------------------|
| 1 | Support maximal | Concept nouveau | Explication complète, architecture, patterns, étapes numérotées | Code source |
| 2 | Structure | Concept vu, pas maîtrisé | Pseudocode, signatures, exemples génériques | Implémentation concrète |
| 3 | Indices ciblés | Concept familier | Hints directionnels, questions socratiques | Structure de solution |
| 4 | Autonomie | Révision, consolidation | Validation, review qualitative | Aide proactive |

## Critères de progression

- **1 → 2** : L'apprenant peut expliquer le concept (test Feynman)
- **2 → 3** : Code fonctionnel écrit avec la structure fournie
- **3 → 4** : 2+ tâches similaires résolues avec indices minimaux

## Critères de rétrogradation

- 3+ erreurs consécutives sur le même concept → remonter d'un niveau
- Demande explicite de l'apprenant (`/aide+`)
